Decimal to Binary Conversion.
1: Divide the decimal value
by 2.
2: Note down the remainder and
quotient as shown.
If the number being divided is
even then remainder will 0 and
1 if the number is odd.
3:Divide the quotient by 2 and
repeat step 2.
4: Repeat above mentioned steps
until you get 0 as quotient.
5.Binary code is then obtained
by placing remainders in order
with the first remainder at
rightmost position and last
remainder at leftmost position.
